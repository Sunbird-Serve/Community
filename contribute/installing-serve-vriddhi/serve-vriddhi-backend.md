# SERVE-Vriddhi Backend

### Need Microservice

Clone the backend repo&#x20;

```
git clone https://github.com/Sunbird-Serve/sunbird-serve-need.git
```

Run below command in bash terminal

```
./gradlew bootRun --args='server.port=9000'
```
